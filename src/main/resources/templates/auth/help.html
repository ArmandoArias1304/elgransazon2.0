help Preguntas Frecuentes

Encuentra respuestas a las preguntas más comunes sobre el sistema

help_outline

¿Cómo puedo recuperar mi contraseña?

Contacta con tu supervisor o administrador del sistema para que pueda restablecer tu contraseña. Por motivos de seguridad, solo los administradores pueden realizar cambios en las credenciales de acceso.
help_outline

¿Qué hago si no puedo iniciar sesión?

Verifica que estés ingresando correctamente tu usuario y contraseña. Asegúrate de que las mayúsculas estén correctas y que no tengas la tecla Bloq Mayús activada. Si el problema persiste, contacta al soporte técnico.
help_outline

¿Qué funciones tengo según mi rol?

Cada rol tiene permisos específicos:
• Administrador: Acceso completo al sistema
• Mesero: Gestión de pedidos y propinas
• Cajero: Cobros y reportes de ventas
• Chef: Gestión de cocina y preparación de pedidos
• Delivery: Gestión de entregas a domicilio
help_outline

¿Cómo actualizo mi información de perfil?

Una vez dentro del sistema, accede a la sección "Mi Perfil" desde el menú principal. Sin embargo, algunos datos sensibles como salario y rol solo pueden ser modificados por un administrador.
help_outline

¿Qué navegadores son compatibles con el sistema?

El sistema es compatible con las versiones más recientes de Google Chrome, Mozilla Firefox, Microsoft Edge y Safari. Recomendamos usar Google Chrome para una mejor experiencia.
help_outline

¿El sistema funciona en dispositivos móviles?

Sí, el sistema cuenta con diseño responsive y se adapta a tablets y smartphones. Puedes acceder desde cualquier dispositivo con conexión a internet.
help_outline

¿Cómo puedo ver mis reportes y estadísticas?

Desde el dashboard principal, selecciona la opción "Reportes" para ver tus estadísticas personales, incluyendo ventas, propinas y desempeño. Los datos se actualizan en tiempo real.
help_outline

¿Qué hago si encuentro un error en el sistema?

Reporta cualquier error o comportamiento inesperado al equipo de soporte técnico. Incluye capturas de pantalla y describe los pasos que realizaste antes de encontrar el error.
